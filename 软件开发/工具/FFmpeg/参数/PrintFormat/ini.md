## ini
```ad-info
title: 此格式为 [[FFprobe]] 中 [[-of]] 参数输出的可选格式
```

### 默认输出示例
```bash
[format]
filename=output.mp4
nb_streams=1
nb_programs=0
format_name=mov,mp4,m4a,3gp,3g2,mj2
format_long_name=QuickTime / MOV
start_time=0.000000
duration=10.234000
size=15797948
bit_rate=12349382
probe_score=100
[format.tags]
major_brand=isom
minor_version=512
compatible_brands=isomiso2avc1mp41
encoder=Lavf59.10.100
```
