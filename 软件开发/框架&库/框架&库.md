## 框架&库

### 框架
```dataviewjs
dv.table(
	["名称","语言"],
	dv.pages('#框架')
	.filter(lib => lib.file.path != dv.current().file.path)
	.sort(lib => lib.file.name)
	.map(lib => [lib.file.link, lib.Language])
);
```

### 库
```dataviewjs
dv.table(
	["名称","语言"],
	dv.pages('#库')
	.filter(lib => lib.file.path != dv.current().file.path)
	.sort(lib => [lib.Language, lib.file.name], 'asc')
	.map(lib => [lib.file.link, lib.Language])
);
```
