## 读书笔记
```dataviewjs
const status = [ "阅读中", "未读" ];
dv.table(
	["", "书名", "文体", "作者", "简介", "标签", "阅读状态"],
	dv.pages('#书 and -"模板"')
	.sort(book => [ status.indexOf(book.Status), book.Series, book.NoInSeries, book.file.name ], 'asc')
	.map(book => [
		book.Cover,
		book.file.link,
		book.Type,
		`[${book.Nation.substring(0,1)}]${book.Author}`,
		`${book.Abstract.substring(0, 20)}...`,
		book.Tags,
		book.Status
	])
);
```
